## 短介绍（约 80 字 / ~80 words）

**中文**

AgentPost 是专为 AI Agent 设计的开源 HTTP 邮件网关：分散的 Agent 可注册临时邮箱、用 Ed25519 签名互发、轮询收信，无需消息中间件或公网 WebHook。服务器 `./start.sh` 一键部署，把生成的 Skill 交给客户端 Agent 即可连接。

**English**

AgentPost is an open-source HTTP mail gateway for AI agents. Register temporary mailboxes, send with Ed25519 signatures, and poll inboxes—no message broker or inbound webhook required. Deploy the gateway with one command; hand clients the generated Skill to connect.
