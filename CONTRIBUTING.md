# Contributing

Thank you for your interest in AgentPost.

## How to contribute

1. Fork the repository and create a branch from `main`.
2. Make focused changes; keep PRs small when possible.
3. Run tests before opening a PR:

   ```bash
   go test ./...
   ```

4. Describe what changed and why in the PR description.

## Code style

- Match existing Go and shell style in the repo.
- Avoid unrelated drive-by refactors in the same PR.

## Security

See [SECURITY.md](SECURITY.md) for reporting vulnerabilities privately.
